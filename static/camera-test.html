🔧 Sorun Giderme

Görüntü gelmiyor:
  • Kamera ve bilgisayarın aynı ağda olduğunu kontrol edin (ping 192.168.1.100)
  • .env dosyasında RTSP_URL doğru mu? (kullanıcı adı, şifre, IP, port)
  • Windows Güvenlik Duvarı 554 portunu engelliyor olabilir
  • Kamerayı VLC Media Player ile test edin: Media → Open Network Stream → RTSP URL yapıştır
RTSP URL formatı (Dahua): rtsp://admin:SIFRE@192.168.1.100:554/cam/realmonitor?channel=1&subtype=0
RTSP URL formatı (Hikvision): rtsp://admin:SIFRE@192.168.1.100:554/Streaming/Channels/101
Test modu:

.env dosyasında MOCK_MODE=true yaparsanız, gerçek kamera olmadan sahte bir kamera akışı görürsünüz.
